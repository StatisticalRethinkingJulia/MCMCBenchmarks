Linear Ballistic Accumulator (LBA)

  • Model
\[ \tau \sim TNormal(.4,.1,0,y_{min}) A \sim TNormal(.8,.4,0,\infty) k \sim TNormal(.2,.3,0,\infty) \v \sim Normal(0,3) (t,c) \sim LBA(A,b,v,s,\tau)\]

where

\[t = y_i - t_{er} b = A + k s = 1 LBA(A,b,v,s,\tau) = f_c(t)\prod_{j \neq c} (1-F_j(t)) f_c(t) = \fract{1}{A} \left[-v_c \Phi\left( \fract{b-A-tv_c}{ts} \right) + \phi\left( \fract{b-A-tv_c}{ts} \right) + + v_c \Phi\left( \fract{b-tv_c}{ts} \right) + s \phi\left( \fract{b-tv_c}{ts} \right) \right] F_c(t) = 1 + \fract{b-A-tv_i}{A} \Phi\left \fract{b-A-tv_c}{ts} \right) - \fract{b-tv_i}{A} \Phi\left \fract{b-tv_c}{ts} \right) + \fract{ts}{A} \phi \left(\fract{b-A-tv_c}{ts} \right) - \fract{ts}{A} \phi \left(\fract{b-tv_c}{ts} \right) Y = {y_1,...,y_n} y_{min} = min{Y}\]
